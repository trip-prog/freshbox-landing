# FreshBox — доставка готового питания

Демо-лендинг для портфолио: сервис доставки готовых рационов с продуктовой механикой, приближенной к веб-приложению.

**Live:** https://trip-prog.github.io/freshbox-landing/

## Фичи

- **Конструктор рациона** — цель, слайдер калорийности, приёмы пищи, длительность; цена, план и скидка пересчитываются на лету
- **Меню дня** — табы завтраки/обеды/ужины/перекусы, карточки блюд с ккал и БЖУ
- **Календарь питания** — интерактивный месяц: клик по дню показывает меню, любой день можно пропустить и вернуть
- **Тарифы** — Balance / Fit / Power, выбор плана настраивает конструктор
- **Оформление подписки** — модалка со сводкой заказа, валидацией и маской телефона

## Стек

Ванильные HTML / CSS / JS — без сборки и зависимостей. Шрифт Onest (Google Fonts), фото Unsplash, иконки — инлайн-SVG (Lucide).

Доступность: aria-атрибуты на табах/модалке/календаре, видимые фокус-состояния, `prefers-reduced-motion`, контраст AA.

## Запуск

Любой статический сервер, например:

```bash
python -m http.server 8757
```
